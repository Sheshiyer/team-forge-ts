# TeamForge Visual Brief

TeamForge is not a generic SaaS dashboard brand. It is a founder mission-control
desktop app with a Star Trek LCARS shell and a systems-console attitude.

## Core Traits

- Mission-control, not friendly productivity
- Structured, segmented, and directional
- High contrast, dark space canvas
- Orange as the primary energy signal
- Cyan, tan, and lavender as disciplined secondary signals
- Precise and reductive, not ornamental

## Color Priorities

- Background: near-black space tones
- Primary: LCARS orange
- Secondary: LCARS cyan
- Support: LCARS tan and lavender
- Avoid: purple-led palettes, soft gradients, generic SaaS blues

## Form Language

- Strong silhouette
- Segmented bands, elbows, rails, apertures, and command rings
- Geometric reduction before texture
- Flat or near-flat icon logic for small sizes
- No text in app icons or route glyphs

## Anti-Patterns

- Generic AI sparkles
- Browser-extension style badge icons
- Liquid chrome or glossy 3D for the main dock icon
- Recraft/Flux-first experimentation for TeamForge icons
- Dense linework that collapses at 16px

## Asset Classes

- Dock icon:
  full brand tile with strongest silhouette
- Tray glyph:
  monochrome simplified mark derived from the dock icon geometry
- Route pictograms:
  modular small glyph family for app navigation
- Campaign board:
  structured release or README collateral using the same mark system
